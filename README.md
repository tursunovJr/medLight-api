# medLight-api
### 1) Цель проекта: Возродить курсовой проект по курсу "Базы данных", переделав его архитектуру.
#### 2) Функциональные требования: 
  + Архитектура - REST API
  + Фреймворк - Flask
  + ORM для БД - SQLArchemy
  + Frontend - HTML, CSS, JS, Bootstrapt v5
